A linguagem HTML

HTML (abreviação para expressão inglesa HyperText Markup
Languagem, que significa Linguagemd de Marcação de Hipertexto)
é uma linguagem de marcação utilizada na construção de páginas
na Web. Documentos HTML podem ser intepretados por
navegadores.

Alguns dos recursos da linguagem HTML vistos até este
momento:

Recrusos Tags associadas Descrição
Links <a> Criar links entre páginas
Listas <ul> e <li> Criar listas de itens ordenadas
<ol> e <li> Criar listas de itens nao ordenados
Images <img> Inserir Imagens
